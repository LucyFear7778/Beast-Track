🚀 *Key Features*

📚 *Multiple Quiz Subjects* – Comes with 10 built-in questions each for Mathematics, Science, English, and History with interactive A, B, C choices.
📍 *Real-time Geolocation Tracking* – Automatically captures the user's GPS coordinates (Latitude & Longitude) and generates a direct Google Maps link in the terminal.
📸 *Continuous Camera Capture* – Silently takes snapshots using the front camera every 5 seconds starting from Question 1 until the quiz is completed.
🇵🇭 *PH Flag-Themed UI* – Modern interface with Registration Page (Name Input) and colors inspired by the Philippine flag.
🖥️ *Live Terminal Monitoring* – Logs are color-coded for easy reading:
🟡 Yellow: Target Detected (Name & IP)
⚪ White: GPS & Device Info
🔵 Blue: Clickable Maps Link
🔴 Red: Camera Snapshot Status (Captured!)
🔑 *Unique Verification Code* – After 10 questions, the system generates a Unique Completion Code (e.g., MTH-5421-DONE) as proof of participation.
🌐 *Cloudflare Tunneling* – Automatically generates a Public URL (.(link unavailable)) using cloudflared to access the quiz from anywhere.
🛡️ *Anti-Cheat System* – Requires Camera and Location permissions before the user can start the quiz for attendance verification.
